# NPMJS Website

## Objective

Learn about the NPMJS website and packages.

## Benefits

All kinds of packages (external modules) can be downloaded from the NPMJS website and used in your projects.

## Complete these tasks

1. Visit the NPMJS website.
2. Search for any pre-made code you're interested in. (Example: cats, animation, movies, buttons, music, etc.)
3. Click on a package for more information about it.
4. In `script.js`, create an array with 2 strings.
5. Let the first string explain the NPMJS website.
6. Let the second string explain an interesting package you found and when it was published.
7. Add `console.log` to display the array.
8. Run `script.js`.

## More Information

- NPM stands for Node Package Manager. JS stands for JavScript.
- The modules in the NPMJS website are known as packages.
- [NPMJS website](https://www.npmjs.com/)

## Usage Tips

- When searching in the NPMJS website, a list of popular results may popup, or click on the `Search` button to view all results.
- The `Readme` tab shows package details.
- The `Code` tab (if available) shows package code.

## Hints

- Packages that were published years ago may or may not work anymore.

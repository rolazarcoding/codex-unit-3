# Exports & Imports

## Objective

Learn what JavaScript exports and imports are.

## Benefits

Exports and imports allow code to be modular, more readable, and easier to reuse.

## Complete these tasks

1. In the terminal, navigate to this level folder.
2. Run `index.js`.
3. Read `index.js` and observe that it imports code from separate files.
4. Read the other `.js` files and observe that their code is exported.
5. In `index.js`, add `console.log` to display a message.
6. Let the message explain the benefits of exporting and importing code.
7. Run `index.js`.

## More Information

- `export` allows code to in one file to be accessible by other files.
- `import` accesses code from another file.
- Exports and imports are a way to split code into separate files or modules so parts of a program can be reused.

## Usage Tips

- Use the `node` command to run a JavaScript file.

## Hints

- The `.js` extension is not necessary to run files. Example: `node my-file`

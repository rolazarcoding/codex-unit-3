# Dependencies

## Objective

Identify project dependencies.

## Benefits

Projects provide a list of dependencies (packages) that are needed to run.

## Complete these tasks

1. In the terminal, navigate to this level folder.
2. Use `npm` to initialize a new project.
3. Install `chalk`.
4. In `script.js`, use `chalk` to display a message.
5. Let the message be color cyan and explain what dependencies are.
6. Run `script.js`. You may need to fix a module error.
7. View `package.json`.
8. Observe the `"dependencies"` section and that `"chalk"` is there.

## More Information

- `"dependencies"` in `package.json` are packages your project needs to run.
- Your project imports dependencies and uses them.

## Usage Tips

- To use `import`, in `package.json` set `"type": "module"`.

## Hints

- The `chalk` package exports an object with various properties.

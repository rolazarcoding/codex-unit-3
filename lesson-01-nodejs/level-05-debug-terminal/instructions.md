# Debug Terminal

## Objective

Learn how to use the JavaScript Debug Terminal.

## Benefits

The JavaScript Debug Terminal can help you find bugs in NodeJS code.

## Complete these tasks

1. Place a breakpoint or the `debugger` command in `script.js`.
2. In VS Code, open a JavaScript Debug Terminal.
3. Navigate to the folder for this level.
4. Run the script. It should stop at the breakpoint.
5. Use the debugger controls to play the script.
6. Use `console.log` to display a message.
7. Let the message explain how to use the JavaScript Debug Terminal to start the debugger.
8. Run `script.js`

## More Information

- Use the JavaScript Debug Terminal to easily engage the debugger.
- [JavaScript Debug Terminal](https://code.visualstudio.com/docs/nodejs/nodejs-debugging#_javascript-debug-terminal)

## Usage Tips

- The default terminal (like Git Bash) may not engage the debugger.
- To open the JavaScript Debug Terminal, do a web search for `How to open the JavaScript debug terminal in VS code`.

## Hints

- Click next to a line number in the code to set a debugger breakpoint.
